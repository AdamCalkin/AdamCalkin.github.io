HTML, CSS & DOM

The difference of HTML asnd CSS.
HTML is the foundations for the website and the directions on what you want it to do.

CSS is the styling in the site bring the color and brghtness to the site.


The DOM is an acronym for Document Object Model.
It is how a devloper and or anyone can see the code for the site to edit and see works in it without editing the actual site,
it can also be used for fun editing random websites to say what you like.

The DOM when edited, doesnt effect the actual website because when the site is viewed it is pulled from the server and displays on your device,
editing the site on your local device only edits the local website.
