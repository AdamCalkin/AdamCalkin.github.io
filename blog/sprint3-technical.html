HTML, CSS & DOM

The difference of HTML asnd CSS.
HTML is the foundations for the website and the directions on what you want it to do.
CSS is the styling in the site bring the color and brghtness to the site.

What is the DOM?
The DOM is an acronym for Document Object Model.
It is how a devloper and or anyone can see the code for the site to edit and see works in it without editing the actual site,
it can also be used for fun editing random websites to say what you like.
The DOM when edited,  doesnt effect the actual website because when the site is viewed it is pulled from the server and displays on your device,
editing the site on your local device only edits the local website.

Explain control flow and loops.
A control flow is the order of individual statements, instructions and functions for a program to execute.
For example,  everyone usally has a rough set of instructions of what they are doing that morning or day.
For me I have waking up,  checking the time,  getting ready for the day,  having a small breakfast then catching a lift into the town to the station.
From then walking to EDA, after that its just catching the bus home.
That is my controled flow that is executed everyday all in a daily loop just like any control flow going in a loop.

Difference between arrays and objects.
Arrays are variables that store multiple values into a single variable, such as when creating a variable
if it is created with many values in it for example names, it is an array.
Objects are variables that have propertries about that specific variable,
such as when describing a car it has a model number, type of vehical, colour and so on.
So when desscribing this car the variable can be called car and in the variable it has values as the other important properties of the car.
The clear difference between these 2 variables is that one (array) defines 1 or many values for a variable
and the other (objects) detailing the properties of the variable to go into more detail.

What are functions?
Functions are blocks of code that are designed to perform a specific task when ran. The function is called when something calls it to run.
A function is definied with a keyword to call the function, which can also have other next to it in parentheses other variables to be defined in this function.
Functions can have in them variables defineing one thing then also returning to a different variable.
